# 手机蓝牙扩音器网页

这个网页会调用手机麦克风，并把声音实时播放到手机当前音频输出设备。  
当手机已连接蓝牙音箱且系统输出选为蓝牙时，就能实现扩音器效果。

## 使用方法

1. 用 HTTPS 或 localhost 打开网页（浏览器麦克风权限要求）。
2. 先在手机系统中连接并切换到蓝牙音箱输出。
3. 点击“开始扩音”，授权麦克风。
4. 用“增益（音量）”调节放大倍数。

## 本地启动（示例）

如果你已安装 Node.js，可在当前目录执行：

```bash
npx serve .
```

然后在手机浏览器中访问对应地址。

## 注意事项

- 手机浏览器通常不支持网页主动切换输出设备，输出路径由系统决定。
- 若出现啸叫（回授），请让手机麦克风远离音箱，或适当降低增益。
